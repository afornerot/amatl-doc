Amatl

This extension is based on Amatl, a binary tool that allows you to convert your Markdown files into HTML and PDF formats.

Transform your CommonMark (also known as Markdown) files into full-fledged documents from the command line.

Why the name amatl? Amate (Spanish: amate [aËˆmate], from Nahuatl languages: Ämatl [ËˆaËmatÍ¡É¬]) is a type of bark paper that has been manufactured in Mexico since pre-Columbian times. It was primarily used to create codices. Source: Wikipedia
